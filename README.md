# **TexFit** – приложение для управления тренировками  

Каждый день после заданного времени формирует из ваших видеофайлов плейлист на текущий день.  
Есть возможность **зациклить** любое упражнение прямо внутри видео (например, тренер — приседает 5 минут, а вам нужно 10).  
Либо, наоборот, **сократить** длительность упражнения (видео — приседания 20 минут, а нужно 10).  
Плеер будет крутить этот отрезок столько, сколько задано.  

**Самое главное** – поскольку каждому нужна разная нагрузка, решено отталкиваться не от количества повторений, а от **длительности выполнения** (10 приседаний за 30 секунд). Для этого в плеере есть **секундомер**, которым можно заранее измерить и настроить нужный промежуток.  

Приложение **специально** написано для **устаревших** телефонов и планшетов, чтобы дать им шанс прослужить дольше (Android 7.0 – Android 15).  

---  

## Самое основное  

- **Импорт видео**  
  - Закидываем одно или несколько видео‑тренировок в формате `.mp4` в любую папку.  
  - Выбираем эту папку через **[+]** (кнопка *Добавить*), затем нажимаем кнопку **Обновить**.  

- **Работа с интерфейсом**  
  - При работе с файлами можно пользоваться ниспадающими списками: достаточно ввести значение один раз, а затем просто выбирать его.  

- **Настройка сеансов**  
  - В таблице задаём **номер** и **имя** сеанса (например, `1 Утро`, `2 День`).  

- **Создание упражнений**  
  - Придумываем номер и название упражнения (например, `01 Зарядка`, `02 Приседания`).  
  - Номер — очередность выполнения: сначала Зарядка, затем Приседания.  

- **Визуальная индикация**  
  - Правильно заполненная строка подсвечивается **зелёным**; незаполненная или исключённая из выборки — **красным**.  
  - Благодаря **нумерации**, упражнения автоматически сортируются в нужном порядке.  

- **Работа с несколькими файлами упражнения**  
  - Если для одного упражнения **несколько** файлов (например, 3 варианта приседаний), задаём номер файла для очередности:  
    `001 приседание1.mp4`, `002 приседание2.mp4`, `003 приседание3.mp4`.  
  - Как только количество файлов для определённого упражнения  закончится, цикл начнется заново с **первого** файла.  

- **Таймер запуска**  
  - Внизу в поле установки времени задаем **время**, после которого будет срабатывать запуск (при первом открытии приложения в этот день).  

- **Просмотр видео**  
  - Нажатие на превью открывает видео в режиме паузы; для **воспроизведения/паузы** нажимаем в любом месте экрана.  
  - На превью внизу можно **галочкой** отметить видео просмотренным. Ни на что не влияет, просто для удобства.
